# 重新设计方案：专业程序员风格文档网站

## 1. 问题分析
- 当前布局：左侧菜单和主内容区域重叠
- 视觉效果：不够专业，缺乏层次感
- 参考图片显示：清晰的左右分栏布局，左侧导航固定，右侧内容区域有适当间距

## 2. 新布局设计方案

### 2.1 整体布局结构
```
+---------------------+---------------------------+
|                     |                           |
|    左侧导航栏       |      主内容区域           |
|   (固定宽度260px)   |   (自适应宽度，留出间距)  |
|                     |                           |
+---------------------+---------------------------+
```

### 2.2 CSS关键调整
1. **左侧导航栏**：
   - 固定位置：`position: fixed; top: 0; left: 0; height: 100vh`
   - 宽度：`width: 260px`
   - 背景：浅灰色 `#f8f9fa`
   - 边框：右侧边框 `border-right: 1px solid #e1e5e9`
   - z-index: 1000（确保在最上层)

2. **主内容区域**：
   - 使用 `margin-left: 260px` 避免重叠
   - 添加内边距：`padding: 24px 40px`
   - 最大宽度限制：`max-width: 1000px`
   - 响应式：移动端时导航栏隐藏，通过汉堡菜单切换

3. **导航项样式**：
   - 当前选中项：蓝色背景 + 左侧蓝色竖条
   - 悬停效果：轻微阴影 + 背景变色
   - 层级缩进：子菜单适当缩进

### 2.3 专业程序员风格增强
- **代码块**：使用Fira Code字体，添加行号
- **标题**：加粗+蓝色下划线，突出重点
- **卡片**：用于重要信息提示，带阴影效果
- **颜色方案**：主色 `#2088ff`（专业蓝色），辅助色 `#6c757d`（灰色）
